## Project Objective

The objective of this project is to evaluate the functional quality and usability
of key user flows on Amazon web platform, focusing on authentication, product search,
and product detail pages.

The goal is to identify functional defects, usability issues, and inconsistencies
that could impact the user experience.

---

## In Scope

The following areas are included in this testing cycle:

### Authentication
- Login page UI
- Valid login
- Invalid login
- Error messages
- Password visibility toggle

### Search Functionality
- Search input field
- Search suggestions
- Search results relevance
- Filters behavior

### Product Detail Page
- Product title
- Price visibility
- Image gallery
- Add to cart button
- Product description section

---

## Out of Scope

The following areas are excluded:

- Payment processing
- Order placement
- User account management
- Prime subscription features
- Backend services

---

## Test Environment

- Platform: Web
- Device: Desktop
- Browser: Google Chrome
- OS: Windows 10
- Internet: Stable broadband connection

---

## Test Approach

Testing will be conducted manually using functional and exploratory techniques.
Test cases will be documented and executed, bugs will be reported with evidence,
and a final summary report will be generated.
